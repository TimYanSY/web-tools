Dynamically Generated Pages

Ways to generate a web page:
  • Statically served HTML files
  • Dynamically templated HTML files
  • Dynamically generated HTML content
  • Dynamically generated Single Page Applications
  • Static Single Page Applications

This file (index.html) is a static file, as is the related CSS file it loads. Often a server will take a request and output HTML content that didn't come from an HTML file, it is the output of a program or function, and is generally known as dynamic content
